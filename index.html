How to Deploy Your Portfolio

  1. Download the files using the button above
  2. Create a new repository named username.github.io
  3. Upload all the downloaded files to your repository
  4. Enable GitHub Pages in your repository settings
  5. Your portfolio will be live at https://username.github.io
